## 글로벌 확장 가능성 분석

### 결론: **가능하다. 단, DESTINO만의 앵글이 있다.**

### 왜 가능한가?

#### 1. 시장이 폭발적으로 성장 중
- 글로벌 $39억 → $90억 (2030), CAGR 20%
- 인도만 CAGR 49% — 아시아가 성장 엔진
- 코로나 이후 디지털 점술 수요 영구 상승

#### 2. DESTINO의 "교차점"은 글로벌에서 유일한 포지션
| 경쟁사 | 제공하는 것 | DESTINO |
|--------|-----------|---------|
| Co-Star | 서양 점성술만 | 동양 사주 x 서양 점성술 x 수비학 |
| 점신/포스텔러 | 동양 사주만 | 동서양 교차 분석 |
| The Pattern | 성격 분석만 | 교차점 = 여러 문명이 같은 답 |

**"Cross-Civilization Destiny Analysis"** — 이 컨셉은 전 세계 어디에도 없다.

#### 3. 영어권에서 사주는 "미지의 체계"
- 서양 유저에게 Chinese/Korean astrology는 **새롭고 흥미로운 콘텐츠**
- 별자리만 보던 유저에게 "너의 별자리와 동양 사주가 같은 답을 냈다"는 강력한 훅
- TikTok에서 "Chinese zodiac" 관련 콘텐츠 조회수 수십억

#### 4. 포스텔러가 이미 증명
- 해외 MAU 40만 (전체의 28%)
- 일본, 미국, 인도, 필리핀, 싱가포르 진출
- 적극적 마케팅 없이 오가닉 성장

### 글로벌 확장 로드맵

#### Phase 1: 영어 버전 (3개월)
- 영어 UI + 영어 해석 텍스트
- "Your Eastern and Western charts agree" 메시지
- TikTok/Instagram Reels 바이럴 콘텐츠
- 타겟: 미국 18-35세 여성 (Co-Star 유저와 겹침)

#### Phase 2: 일본어 버전 (6개월)
- 일본은 운세 문화가 한국만큼 강함
- 사주 대신 "四柱推命" (시주추명) — 일본에서 이미 인기 있는 체계
- 타겟 시장: 1,500만 포스텔러 유저 풀과 겹침

#### Phase 3: 동남아 + 인도 (12개월)
- 인도: CAGR 49%, 가장 빠르게 성장하는 시장
- 인도 Vedic astrology + Western astrology + Numerology 교차 분석
- 동남아: 중국계 인구 → 사주 문화 존재

### 글로벌 확장의 핵심 리스크

| 리스크 | 수준 | 대응 |
|--------|------|------|
| 문화 적응 | 중 | 각 문화권의 운세 체계를 교차 분석에 추가 |
| 현지 경쟁 | 중 | "교차점"이라는 유일한 포지셔닝으로 차별화 |
| 결제 인프라 | 낮 | Stripe 글로벌 결제 |
| AI 해석 품질 | 낮 | Claude API는 다국어 지원 |
| 마케팅 비용 | 높 | 바이럴 콘텐츠로 CAC 최소화 필요 |
