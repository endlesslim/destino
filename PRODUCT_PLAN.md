### 유저가 돈을 내는 이유 (심리)
1. **"나만을 위한 분석"** — 범용 운세가 아닌 개인화된 리포트
2. **검증 욕구** — 내가 느끼는 것이 맞다는 확인
3. **궁합/관계** — 상대방과의 관계 분석이 가장 큰 결제 동기
4. **의사결정 보조** — 이직, 결혼, 투자 타이밍에 "근거" 필요
5. **일상 리추얼** — 매일 확인하는 습관 형성

### 유저가 싫어하는 것
1. 광고 도배 (점신 최대 불만)
2. 뻔한 AI 생성 답변 (앱마다 똑같은 내용)
3. 과금 유도가 노골적 ("이 부분은 유료입니다"로 끊기)
4. 정확도 의심 — 근거 없는 막연한 표현

## 상품 구조

### Tier 1: 무료 미리보기 (Free Preview)
**목적**: 후킹. "이 서비스 뭔가 다르다"를 5초 안에 증명

**제공 내용**:
- 수렴률 퍼센트 (애니메이션 카운터)
- 아키타입 이름만 ("타고난 지휘관")
- 동서양 원소 비교 (土 ↔ Water)
- 오행 밸런스 차트
- **나머지는 블러 처리** → "전체 리포트 보기"

**전환 유도**:
- 블러된 리포트가 얼마나 풍성한지 보여주기
- "이미 847명이 리포트를 받았습니다" 사회적 증거
- 한정 가격 표시: "~~₩19,800~~ → ₩9,900 (첫 분석 50% 할인)"

---

### Tier 2: 전체 리포트 (₩9,900 일회 결제)
**핵심**: 이걸 받으면 "10만원짜리 사주카페 갈 필요 없다"는 느낌

**리포트 구성 (8섹션)**:

#### 1. 교차점 요약 (Overview)
- 수렴률 + 의미 설명
- 아키타입 + 상세 설명 (3-4문장)
- "당신의 교차점 한 줄 요약"

#### 2. 사주 분석 (四柱)
- 연주/월주/일주 (시주는 시간 입력 시)
- 천간 성격 프로파일 (이미지 비유 포함)
- 오행 밸런스 차트 + 부족한 오행 보완법
- 대운 흐름 (10년 단위 운의 흐름) ← **핵심 유료 콘텐츠**

#### 3. 서양 점성술 (Astrology)
- 태양궁 상세 프로파일
- 원소/모달리티 해석
- 지배 행성의 의미
- 올해 트랜짓 운세 ← **시즌별 업데이트 가능**

#### 4. 수비학 (Numerology)
- 생명경로수 상세 해석
- 운명수 (이름 기반)
- 올해의 개인년수 (Personal Year Number) ← **매년 바뀜 → 재구매 동기**
- 행운의 숫자/날짜

#### 5. 교차점 분석 (The Crosspoint) ← **DESTINO만의 핵심**
- 일치하는 특성 상세 (각 체계가 왜 같은 답을 내렸는지)
- 체계간 상생/긴장 관계 설명
- 교차점이 의미하는 당신의 "진짜 본질"

#### 6. AI 맞춤 해석 (Claude 생성)
- 3개 체계 결과를 종합한 자연어 해석문
- "사주카페 선생님이 말해주듯" 톤
- 구체적 조언 포함 (직업, 관계, 건강)

#### 7. 궁합 미리보기 (Compatibility Teaser)
- "당신과 잘 맞는 사주/별자리 조합"
- 상대방 생년월일 입력 → 궁합 분석 (추가 ₩4,900) ← **업셀 포인트**

#### 8. 올해의 운세 (Annual Forecast)
- 사주 기반 월별 운세 키워드
- 별자리 트랜짓 기반 분기별 포인트
- 행운의 달 / 조심할 달

**리포트 형태**:
- 웹 페이지 (고유 URL, 24시간 접근)
- PDF 다운로드 가능 (프리미엄 레이아웃)
- 카카오톡/인스타 공유 카드 자동 생성

## 핵심 설계 원칙

1. **"3개 문명이 같은 답"이 모든 페이지에 반복** — 이것이 유일한 차별점
2. **블러 전략** — 무료로 "와 이거 신기하다"를 느끼게 한 후 결제 유도
3. **구체적 근거 제시** — "~일 수 있습니다" 금지. "3개 체계가 동시에 가리킵니다" 단정
4. **광고 제로** — 경쟁사 최대 불만 요소를 원천 차단
5. **결과물의 물리적 가치** — PDF, 공유 카드 = "내가 뭔가를 받았다"는 느낌
6. **재구매 동기 내장** — 개인년수(매년 변경), 궁합(상대마다 다름), 계절 운세
